# Pet-project | Simple Car Gallery

A small interactive project designed to practice smooth UI animations and clean data management. Hovering over a car name triggers a preview of the corresponding vehicle image with a custom spring-like animation.

### Features
* **UI:** Smooth image reveals triggered by hover effects using custom `cubic-bezier` timing functions.
* **Architecture:** All vehicle data (names, image URLs, metadata) is decoupled from the UI and stored in a structured `data.ts` file, making it easy to add or update items.
* **Stack:** Built using React and Vite for a fast development experience.

### Getting Started
To run this project locally:

```bash
# Clone the repository
git clone git@github.com:SlumberingShark/simple-car-gallery.git

# Install dependencies
npm install

# Start the development server
npm run dev
```
